# Ogame-Bot
This bot, when finished, will allow you to queue up buildings and will build them as soon as possible.

#What is Ogame
Ogame is a browser based resource game. The whole game can be calculated using simple formulas.

#Why
I used to play this game as a kid, probably too much. Recently I was reminiscing with my brother about how much we use to play this game. I realised I could write a script that could play this game very easily.
